塾技 2

食塩水の文章題

▶ 食塩水の文章題の解法手順

【手順①】問題条件をビーカーの図で整理する。

【手順②】各ビーカーの食塩水全体の量および食塩の量を考える。

【手順③】食塩の量または食塩水の濃度について立式する。

\(\text{食塩の量} = \text{食塩水全体の量} \times \dfrac{\text{濃度}(\%)}{100}\)

\(\text{食塩水の濃度}(\%) = \dfrac{\text{食塩の量}}{\text{食塩水全体の量}} \times 100\)

🔥 塾技解説

食塩水の文章題は、とくに難関私立高校を中心に出題されている。押さえるべき点は2つで、食塩水の濃度および食塩の量の求め方だ。

まずは食塩水の濃度。食塩水の濃度は、食塩水全体に対して食塩がどれだけの割合で溶けているかを表す数値で、％を用いて表す。これは何も食塩水に限らず、砂糖水でもオレンジジュースでも同じなんだ。

次に食塩の量。食塩水の文章題の多くは食塩の量に注目して方程式を立てる。求め方は簡単で、食塩水全体の量に濃度をかけるだけ。

食塩の量および食塩水の濃度の求め方、ともにしっかり覚えよう！

【例題】水を加える問題

8%の食塩水200gに、何gかの水を加えて5%の食塩水を作りたい。水は何g加えればよいですか。

加えた水を \(x\) gとし、各ビーカーの食塩の量について立式する。

8% 200g
+
0% \(x\)g
=
5% \((200+x)\)g

食塩の量について：

\(200 \times \dfrac{8}{100} + x \times \dfrac{0}{100} = \dfrac{5}{100}(200+x)\)

両辺を100倍する：

\(1600 + 0 = 5(200+x)\)
\(1600 = 1000 + 5x\)
\(-5x = -600\)
\(x = 120\)
答え: 120g

＋補足：食塩や水を加える問題の考え方

食塩を加えるということ ⇒ 100%の食塩水を加えると考える。

水を加えるということ ⇒ 0%の食塩水を加えると考える。

ポイント 4

濃度に関する問題 Ⅰ

【例題】2種類の食塩水を混ぜる（基本）

4%の食塩水と8%の食塩水を混ぜて、5%の食塩水を200gつくりたい。それぞれ何g混ぜればよいか求めなさい。

指針：全体の重さとふくまれる食塩の重さとで連立方程式をつくる。

4%の食塩水を \(x\) g、8%の食塩水を \(y\) gとする。

4% \(x\)g
+
8% \(y\)g
=
5% 200g

全体の重さの関係から、

\(x + y = 200\) ……(a)

ふくまれる食塩の重さから、

\(0.04x + 0.08y = 200 \times 0.05\) ……(b)

(a), (b)を連立方程式として解くと、\(x = 150, y = 50\)

答え: 4%の食塩水 150g、8%の食塩水 50g

【確認問題 4】

◆(1) 10%の食塩水と7%の食塩水を混ぜて、8%の食塩水を120gつくりたい。それぞれ何g混ぜればよいか求めなさい。

10%の食塩水を \(x\) g、7%の食塩水を \(y\) gとする。

\(x + y = 120\) ……(a)
\(0.10x + 0.07y = 120 \times 0.08 = 9.6\) ……(b)

(a)より \(y = 120 - x\) を(b)に代入：

\(0.10x + 0.07(120 - x) = 9.6\)
\(0.10x + 8.4 - 0.07x = 9.6\)
\(0.03x = 1.2\)
\(x = 40\)

\(y = 120 - 40 = 80\)

答え: 10%の食塩水 40g、7%の食塩水 80g
(2) 20%の砂糖水と50%の砂糖水がある。この砂糖水を混ぜて、30%の砂糖水を300gつくりたい。それぞれ何g混ぜればよいか求めなさい。

20%の砂糖水を \(x\) g、50%の砂糖水を \(y\) gとする。

\(x + y = 300\) ……(a)
\(0.20x + 0.50y = 300 \times 0.30 = 90\) ……(b)

(a)より \(y = 300 - x\) を(b)に代入：

\(0.20x + 0.50(300 - x) = 90\)
\(0.20x + 150 - 0.50x = 90\)
\(-0.30x = -60\)
\(x = 200\)

\(y = 300 - 200 = 100\)

答え: 20%の砂糖水 200g、50%の砂糖水 100g
ポイント 5

濃度に関する問題 Ⅱ

【例題】2種類の食塩水A, B（濃度を求める）

2種類の食塩水A, Bがある。A 50gとB 100gを混ぜると12%の食塩水ができ、A 200gとB 160gを混ぜると14%の食塩水ができるとき、食塩水A, Bの濃度をそれぞれ求めなさい。

指針：Aの濃度を \(x\)%、Bの濃度を \(y\)% として、ふくまれる食塩の重さについて式をつくる。

Aの濃度を \(x\)%、Bの濃度を \(y\)% とする。

1つ目の条件：A 50gとB 100gを混ぜると12%

A 50gにふくまれる食塩 \(50 \times \dfrac{x}{100} = 0.5x\) (g)
B 100gにふくまれる食塩 \(100 \times \dfrac{y}{100} = y\) (g)
12%の食塩水150gにふくまれる食塩 \(150 \times 0.12 = 18\) (g)
よって、\(0.5x + y = 18\) ……(a)

2つ目の条件：A 200gとB 160gを混ぜると14%

A 200gにふくまれる食塩 \(200 \times \dfrac{x}{100} = 2x\) (g)
B 160gにふくまれる食塩 \(160 \times \dfrac{y}{100} = 1.6y\) (g)
14%の食塩水360gにふくまれる食塩 \(360 \times 0.14 = 50.4\) (g)
よって、\(2x + 1.6y = 50.4\) ……(b)

(a), (b)を連立方程式として解くと、\(x = 18, \; y = 9\)

答え: Aの濃度 18%、Bの濃度 9%

【確認問題 5】

◆(1) 2種類の食塩水A, Bがある。A 200gとB 100gを混ぜると5%の食塩水ができ、A 100gとB 500gを混ぜると8%の食塩水ができるとき、A, Bの濃度をそれぞれ求めなさい。

Aの濃度を \(x\)%、Bの濃度を \(y\)% とする。

条件1：A 200g + B 100g → 5%の食塩水300g

\(2x + y = 300 \times 0.05 = 15\) ……(a)

条件2：A 100g + B 500g → 8%の食塩水600g

\(x + 5y = 600 \times 0.08 = 48\) ……(b)

(a)より \(y = 15 - 2x\) を(b)に代入：

\(x + 5(15 - 2x) = 48\)
\(x + 75 - 10x = 48\)
\(-9x = -27\)
\(x = 3\)

\(y = 15 - 2(3) = 9\)

答え: Aの濃度 3%、Bの濃度 9%
(2) A, B 2種類の食塩水が500gずつある。Aから300g、Bから200g取り出して混ぜ合わせたら8%の食塩水ができた。また、残った食塩水を混ぜ合わせたら7%の食塩水ができた。A, Bの濃度をそれぞれ求めなさい。

Aの濃度を \(x\)%、Bの濃度を \(y\)% とする。

条件1：A 300g + B 200g → 8%の食塩水500g

\(3x + 2y = 500 \times 0.08 = 40\) ……(a)

条件2：残りのA 200g + B 300g → 7%の食塩水500g

\(2x + 3y = 500 \times 0.07 = 35\) ……(b)

(a)×3 − (b)×2：

\(9x + 6y = 120\)
\(4x + 6y = 70\)
\(5x = 50\)
\(x = 10\)

\(y = \dfrac{40 - 3(10)}{2} = \dfrac{10}{2} = 5\)

答え: Aの濃度 10%、Bの濃度 5%

【3】次の問いに答えなさい。（◇ポイント3）

◆(1) A, B 2種類のジュースがあり、昨日は合わせて400本売れた。今日売れた本数は、昨日に比べてAが20%、Bが10%それぞれ減り、合わせて330本であった。昨日売れたA, Bのジュースの本数をそれぞれ求めなさい。

昨日売れたAのジュースを \(x\) 本、Bのジュースを \(y\) 本とする。

\(x + y = 400\) ……(a)
\(0.8x + 0.9y = 330\) ……(b)

(a)より \(y = 400 - x\) を(b)に代入：

\(0.8x + 0.9(400 - x) = 330\)
\(0.8x + 360 - 0.9x = 330\)
\(-0.1x = -30\)
\(x = 300\)

\(y = 400 - 300 = 100\)

答え: A 300本、B 100本
(2) ある中学校の生徒会では、毎月アルミ缶を回収している。9月に集めた缶の重さは、2年生と3年生合わせて38kgであった。10月に集めた缶の重さは、3年生の方が2年生よりも6kg多かったが、9月に比べると、2年生は50%増加し、3年生は25%減少していた。10月に2年生と3年生が集めた缶の重さをそれぞれ求めなさい。

9月の2年生を \(x\) kg、3年生を \(y\) kgとする。

\(x + y = 38\) ……(a)

10月は、2年生 \(1.5x\) kg、3年生 \(0.75y\) kg。

\(1.5x - 0.75y = 6\)（3年生の方が6kg多い → ではなく差が6）

問題文「3年生の方が2年生よりも6kg多かった」より：

\(0.75y - 1.5x = 6\) ……(b)

(a)より \(y = 38 - x\) を(b)に代入：

\(0.75(38 - x) - 1.5x = 6\)
\(28.5 - 0.75x - 1.5x = 6\)
\(-2.25x = -22.5\)
\(x = 10\)

\(y = 38 - 10 = 28\)

10月：2年生 \(1.5 \times 10 = 15\) kg、3年生 \(0.75 \times 28 = 21\) kg

答え: 2年生 15kg、3年生 21kg

【4】次の問いに答えなさい。（◇ポイント4）

◆(1) 3%の食塩水と8%の食塩水を混ぜて、5%の食塩水を500gつくりたい。それぞれ何g混ぜればよいか求めなさい。

3%の食塩水を \(x\) g、8%の食塩水を \(y\) gとする。

\(x + y = 500\) ……(a)
\(0.03x + 0.08y = 500 \times 0.05 = 25\) ……(b)

\(0.03x + 0.08(500 - x) = 25\)
\(0.03x + 40 - 0.08x = 25\)
\(-0.05x = -15\)
\(x = 300\)

\(y = 500 - 300 = 200\)

答え: 3%の食塩水 300g、8%の食塩水 200g
(2) 4%の食塩水に食塩を加えて、12%の食塩水を240gつくりたい。4%の食塩水何gに食塩何gを混ぜればよいか求めなさい。

4%の食塩水を \(x\) g、加える食塩を \(y\) gとする。

※食塩は100%の食塩水と考える。

\(x + y = 240\) ……(a)

【5】次の問いに答えなさい。（◇ポイント5）

◆(1) 120gの食塩水Aと80gの食塩水Bがある。この2つの食塩水を混ぜると12%の食塩水ができる。また、食塩水A 30gと食塩水B 80gにふくまれる食塩の量は等しい。食塩水A, Bの濃度を求めなさい。

Aの濃度を \(x\)%、Bの濃度を \(y\)% とする。

条件1：A 120g + B 80g → 12%の食塩水200g

\(1.2x + 0.8y = 200 \times 0.12 = 24\) ……(a)

条件2：A 30gの食塩 = B 80gの食塩

\(0.3x = 0.8y\) ……(b)

(b)より \(x = \dfrac{8}{3}y\)。(a)に代入：

\(1.2 \times \dfrac{8}{3}y + 0.8y = 24\)
\(3.2y + 0.8y = 24\)
\(4y = 24\)
\(y = 6\)

\(x = \dfrac{8}{3} \times 6 = 16\)

答え: Aの濃度 16%、Bの濃度 6%
(2) 容器A, Bに異なる濃度の食塩水が、Aには400g, Bには300g入っている。A, Bからそれぞれ100gずつ取り出して混ぜ合わせると5%の食塩水ができた。次に、A, Bに残った食塩水をすべて混ぜ合わせ、100gだけ水を蒸発させると5.5%の食塩水になった。はじめに容器A, Bに入っていた食塩水の濃度をそれぞれ求めなさい。

Aの濃度を \(x\)%、Bの濃度を \(y\)% とする。

条件1：A 100g + B 100g → 5%の食塩水200g

\(x + y = 200 \times 0.05 = 10\) ……(a)

条件2：残りのA 300g + B 200g → 500g。水を100g蒸発 → 400gで5.5%

食塩の量は蒸発で変わらないので：
\(3x + 2y = 400 \times 0.055 = 22\) ……(b)

(a)より \(y = 10 - x\) を(b)に代入：

\(3x + 2(10 - x) = 22\)
\(3x + 20 - 2x = 22\)
\(x = 2\)

\(y = 10 - 2 = 8\)

答え: Aの濃度 2%、Bの濃度 8%

【6】鉄橋と列車

長さ360mの貨物列車が、ある鉄橋を渡り始めてから、渡り終わるまで1分かかった。また、長さ200mの特急列車が、貨物列車の2倍の速さでこの鉄橋を渡り始めてから、渡り終わるまで25秒かかった。貨物列車の速さと、鉄橋の長さを求めなさい。

貨物列車の速さを毎秒 \(x\) m、鉄橋の長さを \(y\) mとする。

条件1：貨物列車が鉄橋を渡る（列車の長さ＋鉄橋の長さ＝走った距離）

\(60x = 360 + y\) ……(a)

条件2：特急列車（速さ \(2x\) m/s）が鉄橋を渡る

\(25 \times 2x = 200 + y\)
\(50x = 200 + y\) ……(b)

(a) − (b)：

\(10x = 160\)
\(x = 16\)

\(y = 60 \times 16 - 360 = 960 - 360 = 600\)

答え: 貨物列車の速さ 毎秒16m（時速57.6km）、鉄橋の長さ 600m

【7】売上と個数

定価が1個250円のある商品を、A店とB店で販売した。A店では最初から最後まで定価の20%引きで販売した。B店では初め定価で販売したが、途中から定価の半額で販売した。定価の半額で販売した個数は84個であった。A店とB店で販売した商品の個数の合計は690個で、A店、B店それぞれの売上金の総額は同じであった。このとき、A店、B店それぞれで販売した商品の個数を求めなさい。

A店の販売個数を \(a\) 個、B店の販売個数を \(b\) 個とする。

A店の売上単価：\(250 \times 0.8 = 200\) 円

B店は、定価250円で \((b - 84)\) 個、半額125円で84個販売。

\(a + b = 690\) ……(a)

売上が同じなので：

\(200a = 250(b - 84) + 125 \times 84\)
\(200a = 250b - 21000 + 10500\)
\(200a = 250b - 10500\) ……(b)

(a)より \(a = 690 - b\) を(b)に代入：

\(200(690 - b) = 250b - 10500\)
\(138000 - 200b = 250b - 10500\)
\(148500 = 450b\)
\(b = 330\)

\(a = 690 - 330 = 360\)

答え: A店 360個、B店 330個

【問題】食塩水の交換（成蹊高）

容器Aには9%の食塩水が400g、容器Bには4%の食塩水が240g入っている。容器A, Bからそれぞれ \(x\) gの食塩水を取り出し、容器Aから取り出した食塩水は容器Bに、容器Bから取り出した食塩水は容器Aに入れてよくかき混ぜたところ、2つの容器の中の食塩水の濃度が同じになった。このとき、\(x\) の値を求めなさい。

まず、問題条件をビーカーの図で整理する。

新たな容器A：

9% 400g
−
9% \(x\)g
+
4% \(x\)g
=
?% 400g

食塩の量：

\(400 \times \dfrac{9}{100} - \dfrac{9}{100}x + \dfrac{4}{100}x = 36 - \dfrac{1}{20}x\)

新たな容器B：

4% 240g
−
4% \(x\)g
+
9% \(x\)g
=
?% 240g

食塩の量：

\(240 \times \dfrac{4}{100} - \dfrac{4}{100}x + \dfrac{9}{100}x = 9.6 + \dfrac{1}{20}x\)

できあがった容器Aと容器Bの濃度が等しいことより、

\(\dfrac{36 - \dfrac{1}{20}x}{400} \times 100 = \dfrac{9.6 + \dfrac{1}{20}x}{240} \times 100\)

両辺を1200倍する：

\(3\left(36 - \dfrac{1}{20}x\right) = 5\left(9.6 + \dfrac{1}{20}x\right)\)

\(108 - \dfrac{3}{20}x = 48 + \dfrac{5}{20}x\)

\(108 - 48 = \dfrac{5}{20}x + \dfrac{3}{20}x\)

\(60 = \dfrac{8}{20}x = \dfrac{2}{5}x\)

\(x = 150\)
答え: \(x = 150\)

【問題 1】（明治学院高）

濃度4%の食塩水が600g入っている容器がある。ここから食塩水 \(x\) gを取り出したとき、次の問いに答えなさい。

(1) 容器に残っている食塩水に含まれる食塩の量を \(x\) を用いた式で表しなさい。

元の食塩の量：\(600 \times 0.04 = 24\) g

取り出した食塩水 \(x\) gに含まれる食塩：\(x \times 0.04 = 0.04x\) g

残っている食塩の量：

\(24 - 0.04x\) (g)
答え: \((24 - 0.04x)\) g
(2) このあと、容器に \(2x\) gの水を入れてかき混ぜたところ濃度が0.8%になった。\(x\) の値を求めなさい。

水を入れた後の食塩水の量：\((600 - x) + 2x = 600 + x\) g

食塩の量は変わらず \(24 - 0.04x\) g。濃度が0.8%なので：

\(\dfrac{24 - 0.04x}{600 + x} = 0.008\)

両辺に \((600 + x)\) をかける：

\(24 - 0.04x = 0.008(600 + x)\)
\(24 - 0.04x = 4.8 + 0.008x\)
\(19.2 = 0.048x\)
\(x = 400\)
答え: \(x = 400\)

【問題 2】（ラ・サール高）

A, B, Cの容器にそれぞれ5%, 10%, \(x\)%の食塩水がいくらかずつ入っている。AとBの食塩水をすべて混ぜると8%、BとCの食塩水をすべて混ぜると13%、AとCの食塩水をすべて混ぜると11%になる。まず、A, Bに入っていた食塩水の量の比をもっとも簡単な整数の比で表し、そして \(x\) の値を求めなさい。

A, B, Cに入っている食塩水の量をそれぞれ \(a\) g, \(b\) g, \(c\) gとする。

A+B=8% の条件（てんびん法 / 内分比で考える）：

\(0.05a + 0.10b = 0.08(a + b)\)
\(0.05a + 0.10b = 0.08a + 0.08b\)
\(0.02b = 0.03a\)
\(\dfrac{a}{b} = \dfrac{2}{3}\)
答え（比）: A : B = 2 : 3

\(x\)の値を求める：

\(a = 2k, b = 3k\) とおく。

B+C=13% の条件：

\(0.10 \times 3k + 0.01xc = 0.13(3k + c)\)
\(0.3k + 0.01xc = 0.39k + 0.13c\)
\(0.01xc - 0.13c = 0.09k\)
\(c(0.01x - 0.13) = 0.09k\) ……(i)

A+C=11% の条件：

\(0.05 \times 2k + 0.01xc = 0.11(2k + c)\)
\(0.1k + 0.01xc = 0.22k + 0.11c\)
\(0.01xc - 0.11c = 0.12k\)
\(c(0.01x - 0.11) = 0.12k\) ……(ii)

(i) ÷ (ii)：

\(\dfrac{0.01x - 0.13}{0.01x - 0.11} = \dfrac{0.09}{0.12} = \dfrac{3}{4}\)

\(4(0.01x - 0.13) = 3(0.01x - 0.11)\)
\(0.04x - 0.52 = 0.03x - 0.33\)
\(0.01x = 0.19\)
\(x = 19\)
答え: \(x = 19\)
